Privacy Policy for Copy That

Last updated: [12/07/2024]

Copy That is open-source software developed to replace the Windows Explorer copying, moving, and deletion functionality. As an open-source project, we prioritize transparency and respect your privacy. This Privacy Policy explains how we handle data related to the use of this software.

  1. Data Collection
    • No Personal Data Collection: Copy That does not collect, store, or transmit any personal information from users.
    • Optional Analytics (if applicable): If you choose to enable analytics or telemetry, the data collected will be anonymized and used solely for improving the software.
  2. Data Sharing
    • No Sharing with Third Parties: Since we do not collect personal data, we do not share information with third parties.
  3. Open Source and Transparency
    • The source code of Copy That is publicly available at [Copy That Repository]. Users are encouraged to review the code to verify how their data is handled.
  4. User Responsibility
    • Third-Party Libraries/Services: If the software interacts with third-party services (e.g., APIs, plugins), users are responsible for reviewing the privacy policies of those services.
  5. Changes to This Policy
    • This Privacy Policy may be updated periodically to reflect changes in the software or legal requirements. Updates will be documented in the project's repository or website.
  6. Contact Information
    • If you have any questions or concerns regarding this Privacy Policy, don't hesitate to [Contact Us].
